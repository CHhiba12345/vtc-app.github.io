# Image Sources Guide

## Recommended Free Image Sources for VTC Website

### Vehicles Images

- **Unsplash**: https://unsplash.com/s/photos/luxury-car
- **Pexels**: https://www.pexels.com/search/mercedes%20benz/
- **Pixabay**: https://pixabay.com/images/search/bmw/

### Keywords to search:

- "mercedes e class"
- "bmw 5 series"
- "audi a6"
- "tesla model s"
- "luxury sedan"
- "premium vehicle"
- "chauffeur car"

### Driver Images

- **Unsplash**: https://unsplash.com/s/photos/professional-driver
- **Pexels**: https://www.pexels.com/search/chauffeur/

### Service Images

- **Marseille Airport**: Search "marseille airport"
- **Business District**: Search "marseille business"
- **Old Port Marseille**: Search "vieux port marseille"

## Image Specifications

- **Format**: JPG or WebP
- **Vehicle Images**: 800x600px minimum
- **Driver Photos**: 400x400px (square)
- **Service Images**: 1200x800px

## Usage Instructions

1. Download images from these sources
2. Rename them according to the paths in the code
3. Place in the appropriate public/img/ folders
4. Ensure you have rights to use the images commercially
